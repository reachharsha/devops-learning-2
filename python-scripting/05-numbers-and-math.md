# Lesson 05 — Numbers & Math

## Objectives

- Work with `int` and `float`
- Use common operators: `+ - * / // % **`
- Avoid common beginner mistakes

## Concepts

### Operators

- `+` add
- `-` subtract
- `*` multiply
- `/` divide (result is usually a float)
- `//` integer division
- `%` remainder (mod)
- `**` power

## Hands-on Lab

Create `math_demo.py`:

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
```

## Quick Check

1) What is the difference between `/` and `//`?
2) What does `%` compute?
3) What is `2 ** 3`?

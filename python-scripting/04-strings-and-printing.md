# Lesson 04 — Strings & Printing

## Objectives

- Create strings
- Print output
- Use f-strings for readable formatting

## Concepts

### Strings

Strings are text:

```python
message = "hello"
```

### Printing

```python
print("Hello")
```

### f-strings (recommended)

```python
name = "Alice"
age = 30
print(f"{name} is {age} years old")
```

## Hands-on Lab

Create `print_demo.py`:

```python
first = "DevOps"
second = "Python"

print("First:", first)
print("Second:", second)
print(f"Together: {first} + {second}")
```

## Quick Check

1) What is an f-string?
2) What does `print()` do?
3) Which is easier to read: string `+` concatenation or f-strings?
